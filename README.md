# 🛡️ AI Red Team Assistant

AI Red Team Assistant is a web-based security planning tool built using Python and Streamlit.  
It helps generate reconnaissance checklists, OSINT queries, and MITRE-mapped attack vectors for educational and defensive security purposes.

## 🚀 Features

- 📌 Target-based Recon Checklist
- 🔍 OSINT Query Generator (Google Dorks, Shodan-style queries)
- 🎯 MITRE ATT&CK Technique Mapping
- 📝 Report Upload & Download
- 🌙 Dark Mode UI
- ☁️ Deployed using Streamlit Community Cloud

---

## 🛠️ Technologies Used

- Python
- Streamlit
- HTML (Inline Styling)
- CSS (Custom Styling)
- GitHub (Version Control)
- Streamlit Community Cloud (Deployment)
